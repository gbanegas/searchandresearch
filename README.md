searchandresearch
=================

Pre-requisites
-----------
It is necessary to install python >= 2.6

For deabian-based:
    sudo apt-get install python2.6  python-pip

Then install [watchdog|https://pypi.python.org/pypi/watchdog]
    sudo pip install watchdog

Usage
-----
    python main.py

All the logs will be in the \server_1\
